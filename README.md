# NutriFast — Sistema de Registro de Calorias e Jejum

Uma aplicação web robusta desenvolvida para monitoramento de saúde pessoal e gerenciamento metabólico através do controle de ingestão calórica diária e ciclos de jejum intermitente.

## 🚀 Stack Utilizada
- **Framework:** Next.js 14 (App Router) com React Server Components
- **Linguagem:** TypeScript
- **Banco de Dados & Autenticação:** Supabase (PostgreSQL + Supabase Auth)
- **Segurança:** Row Level Security (RLS) nativo em PostgreSQL
- **Estilização:** Tailwind CSS
- **Gráficos:** Recharts
